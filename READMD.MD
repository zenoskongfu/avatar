### Installation & Usage

#### Install
```bash
npm install avatar-generator
```

#### React Usage
```jsx
import { AvatarReact } from 'avatar';

function App() {
  return <AvatarReact str="Example" height={200} />;
}
```

#### Vue Usage
```vue
<script setup>
import { AvatarVue } from 'avatar';
</script>

<template>
  <AvatarVue str="Example" :height="200" />
</template>
```
